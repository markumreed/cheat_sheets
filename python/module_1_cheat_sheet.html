🧩 Variables & Data Types

Use variables to label values you’ll reuse. Python infers the type from the value.

company = "USF Analytics"   # str (text)
employees = 157             # int (whole number)
salary = 72000.50           # float (decimal)
profitable = True           # bool (True/False)
💡
Tip: Python is case-sensitive. Revenue ≠ revenue. Use descriptive names like north_revenue.
🖨️ Printing Output

print() displays values in the Shell.

print("Hello, Business World!")
print("Revenue:", 50000)
print("Profit Margin:", 0.25 * 100, "%")
📝
Note: Use commas to print text and numbers together. Python inserts spaces for you.
➗ Arithmetic Operators

+ − × ÷ for business math.

# + Addition, - Subtraction, * Multiplication, / Division
revenue = 100000
costs   =  75000
profit  = revenue - costs
print("Profit: $", profit)
💡
Tip: For averages, divide by the count: avg = total / n.
#️⃣ Comments

Use comments to explain intent. Python ignores them when running.

# This is a comment explaining the next line
print("Hello!")
📝
Note: Great comments explain why, not just what.
🐞 Debugging in Thonny

Step through code to see values change and find mistakes.

  1. Click 🐞 Debug in Thonny.
  2. Use step controls to execute line by line.
  3. Watch variables in the Variable Inspector.
💡
Tip: If output surprises you, debug. Step to the first “wrong” value.
💡 Common Errors & Fixes
# Missing quotes (SyntaxError)
print(Hello World)          # ❌
print("Hello World")        # ✅

# Mixing text + numbers with +
total = 190000
print("Total: $" + total)   # ❌ TypeError
print("Total: $", total)    # ✅

# Misspelled variable name (NameError)
total_revenue = 100000
print(total_revenu)         # ❌
print(total_revenue)        # ✅
📝
Note: Read error messages—they usually tell you the exact line and type of error.
📊 Business Mini‑Example
# Q1 sales summary
jan = 15000
feb = 18000
mar = 22000

total_q1 = jan + feb + mar
avg_q1   = total_q1 / 3

print("Q1 Total:", total_q1)
print("Q1 Average:", avg_q1)
print("March > January?", mar > jan)
💡
Tip: Turn this into a function later to reuse across quarters.
